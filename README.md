# rose

	Connect 6 servos up to rose petals and then use this to send IR commands to drop the petals off the rose.
 
 	Circuit:
	- Petals are on servos attached to pins 4 through 9
	- The IR interface is on pin 11
 
	commands are:
	- RC5 0xFF629A
	- RC5 0xFF629B
	- RC5 0xFF629C
	- RC5 0xFF629D
	- RC5 0xFF629E
	- RC5 0xFF629F
